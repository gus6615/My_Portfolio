**게임 목표** : 채광을 하고 돈을 벌어 부자가 되는 게임입니다.<br><br>

☞ 펫을 얻어 방치하기 (방치형 콘텐츠)<br>
☞ 랭킹 시스템과 업적 시스템 (유저 간 경쟁 콘텐츠)<br>
☞ 메인 혹은 일일 퀘스트<br>
☞ 출석 및 주말 이벤트<br>
☞ 자동 채광<br>
☞ 업적<br>
☞ 채팅<br><br>
